Project: AWS ETL Pipeline

Business Problem
---------------
Build an ETL pipeline that ingests sales data,
stores it in S3,
transforms it using AWS Glue,
queries it with Athena,
and visualizes KPIs in Power BI.

Architecture
------------
CSV Files
    ↓
Amazon S3
    ↓
AWS Lambda
    ↓
AWS Glue
    ↓
Athena / Redshift
    ↓
Power BI

Tech Stack
----------
Python
AWS S3
AWS Lambda
AWS Glue
Athena
Redshift
Power BI
SQL

Key Learnings
-------------
ETL Design
Data Lake Concepts
AWS Glue Jobs
Athena Querying
Warehouse Analytics
Dashboard Development
